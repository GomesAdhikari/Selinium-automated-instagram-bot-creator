# Instagram Account Creation Script

## Overview

This project provides a Python script for automating the creation of Instagram accounts. It uses **Selenium** for browser automation, allowing for programmatic interaction with the Instagram sign-up page.

**Note**: This script should only be used for legitimate and ethical purposes, such as creating accounts for businesses or research with proper authorization. Use this tool responsibly and in compliance with Instagram's terms of service.

## Prerequisites

Before running this script, ensure that:

- Python 3.x is installed on your system.
- Chrome and **ChromeDriver** are installed and configured in your system's PATH.
- **Selenium** library is installed.
